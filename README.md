# cizelge

## Install

```bash
npm i cizelge
```

## Usage

```jsx
import React, { Component } from 'react'

import { LineChart } from 'cizelge'

const App = () => {
    return <LineChart xAxis={[]} yAxis={[]} sourceName={'line source'} />
}
```
